GOAT Stack Examples

Explore the power of Alpine.js and Tailwind CSS with these interactive examples

Basic Counter

Current count: 

<div x-data="{ count: 0 }">\n  <p>Current count: <span x-text="count"></span></p>\n  <button @click="count++">Increment</button>\n</div>

Toggle Visibility

This content can be toggled!

<div x-data="{ show: false }">\n  <button @click="show = !show">Toggle Content</button>\n  <p x-show="show" x-transition>This content can be toggled!</p>\n</div>

Two-way Binding

Hello, !

<div x-data="{ name: '' }">\n  <input type="text" x-model="name" placeholder="Enter your name">\n  <p>Hello, <span x-text="name || 'stranger'"></span>!</p>\n</div>

Conditional Styling

Count: 

<div x-data="{ count: 0 }">\n  <p>Count: <span x-text="count" :class="{ 'text-green-500': count > 0, 'text-red-500': count < 0 }"></span></p>\n  <button @click="count++">Increment</button>\n  <button @click="count--">Decrement</button>\n</div>

Loading State

<div x-data="{ loading: false }">\n  <button @click="loading = true; setTimeout(() => loading = false, 2000)" :disabled="loading">\n    <span x-show="!loading">Click me</span>\n    <span x-show="loading">\n      <!-- Loading spinner SVG -->\n      Loading...\n    </span>\n  </button>\n</div>
